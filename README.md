# 📰 Fake News Detection

A Machine Learning project that classifies news articles as **Fake or Real** using Natural Language Processing (NLP) techniques.

---

## 🚀 Tech Stack
- Python
- Pandas
- Scikit-learn
- NLTK

---

## 📊 Project Progress
- Day 1: NLP Basics ✅
- Day 2: Dataset Processing ✅
- Day 3: Text Preprocessing ✅
- Day 4: Model Training & Evaluation ✅

---

## 📈 Model Performance
- Accuracy: **98.8%**
- Algorithm: Logistic Regression
- Feature Extraction: TF-IDF

---

## 🔄 Workflow
1. Data Collection (Fake & Real news dataset)
2. Text Preprocessing (Cleaning, Stopwords Removal, Stemming)
3. Feature Extraction using TF-IDF
4. Model Training (Logistic Regression)
5. Evaluation using Accuracy & Confusion Matrix
